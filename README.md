# 26-1_alom_ai_precourse

**아롬 AI 예비대학**은 신입생과 학부생을 대상으로 하는 **컴퓨터비전(CV) 입문 스터디/세미나**입니다.  
Python과 PyTorch를 사용해 이미지 데이터를 다루는 방법을 익히고, 마지막 2주 동안 **미니 프로젝트**로 마무리합니다.

## 목표
- OpenCV를 활용하여 필터링, 엣지 검출, 객체 검출 등 전통적인 컴퓨터 비전 기법을 직접 구현하고 원리를 이해합니다.
- PyTorch를 활용해 CNN 기반 이미지 분류 모델을 학습하고 평가하며, 딥러닝 기반 컴퓨터 비전의 기본 흐름을 익힙니다.
